🛠️ 문제 해결 방법

PHP가 작동하지 않는 경우:

  1. 호스팅 업체에 PHP 활성화 요청
  2. PHP 버전을 7.4 이상으로 설정 요청
  3. 파일 확장자가 .php인지 확인
  4. 파일 권한을 644로 설정

대안 방법:

  1. SSH 접속: 호스팅 업체에 SSH 접속 권한 요청
  2. cPanel/Plesk: 웹 호스팅 제어판의 터미널 기능 사용
  3. Node.js 호스팅: Vercel, Netlify 등 Node.js 전용 호스팅 사용
  4. 정적 빌드: Next.js를 정적 사이트로 빌드하여 업로드

📞 호스팅 업체 문의 사항

다음 사항들을 호스팅 업체에 문의하세요:

  • ✅ PHP 7.4 이상 지원 여부
  • ✅ shell_exec 함수 활성화 가능 여부
  • ✅ Node.js 설치 가능 여부
  • ✅ SSH 접속 권한 제공 여부
  • ✅ 포트 3000 개방 가능 여부
  • ✅ 백그라운드 프로세스 실행 허용 여부

🎯 즉시 확인할 사항
